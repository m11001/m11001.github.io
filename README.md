![](img/10.png)

有很多的数据包

这时候就要自己点击进去慢慢找了

不过不要担心，很容易找到的

找到类似下面这种的数据包

![](img/11.png)

我打马赛克的部分就是我得`key`

![](img/12.png)

注意是 `_key` 后面的值一直到 `&` 结束,长为 **112**

再将我打马赛克的部分复制出来即可

访问 http://xiao-xi.tk/

![](img/13.png)

在箭头处粘贴你刚才复制的`key`的值即可

签到成功截图

![](img/14.png)

## PS

key 值只要葫芦侠没有注销重新登陆，是一直不会改变的，所以记下来以后可以使用
